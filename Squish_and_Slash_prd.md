# Squish & Slash — 게임 기획서

> 말랑한 슬랑이를 자르고, 왁뿌볼을 부수는 ASMR 슬라이스 게임

---

## 1. 게임 개요

| 항목 | 내용 |
|------|------|
| 제목 | Squish & Slash |
| 장르 | 캐주얼 슬라이스 아케이드 |
| 플랫폼 | 모바일 우선 (웹 배포 → 추후 앱) |
| 조작 | 터치 드래그 (한 손가락 슬라이스) |
| 핵심 감성 | 말랑한 촉감 + 만족스러운 소리(ASMR) |
| 레퍼런스 | 후르츠 닌자 + 슬랑이/왁뿌볼 트렌드 |

**한 줄 컨셉**
아래에서 통통 튀어오르는 슬랑이를 손가락으로 슬라이스하고, 왁뿌볼을 부수며 점수를 쌓는다. 단, 딱딱이 얼음은 피해야 한다.

---

## 2. 핵심 메커니즘

### 2.1 기본 루프
1. 화면 아래에서 오브젝트가 위로 튀어오른다 (포물선 운동)
2. 플레이어가 손가락을 드래그해 오브젝트를 가른다
3. 오브젝트별로 다른 연출/소리/점수가 발생한다
4. 놓치거나 잘못 자르면 패널티
5. 제한 시간 또는 생명 소진 시 종료

### 2.2 오브젝트 종류

| 오브젝트 | 행동 | 연출 | 소리 | 점수 |
|----------|------|------|------|------|
| 🟢 슬랑이 | 자른다 | 말랑하게 쭉 갈라짐 + 젤리 파편 | 촉촉한 "찰랑~" | +10 |
| 🟡 왁뿌볼 | 부순다 | 단단하게 쪼개짐 + 왁스 조각 | 둔탁한 "퍽!" | +20 (보너스) |
| 🧊 딱딱이 얼음 | 피한다 | 칼이 튕김 + 화면 흔들림 | 차가운 "캉!" | 콤보 끊김 / 패널티 |

### 2.3 판정
- 드래그 궤적이 오브젝트의 콜라이더를 통과하면 슬라이스 성공
- 한 번의 드래그로 여러 개를 자르면 **콤보** 발생
- 얼음을 자르면 생명 -3 (시작 생명이 3이므로 사실상 즉시 종료)
- 슬랑이/왁뿌볼이 여러 개 튀어오를 때 얼음이 섞여 등장 → 한 번에 쓸어 자르면 위험 (난이도 조절 핵심)

---

## 3. 점수 시스템

### 3.1 기본 점수
- 슬랑이: +10
- 왁뿌볼: +20

### 3.2 콤보 시스템
- 한 드래그에 2개 이상 슬라이스 시 콤보 보너스
- 콤보 배수: 2콤보 ×1.2 / 3콤보 ×1.5 / 4콤보+ ×2.0
- 얼음 슬라이스 또는 슬랑이 놓침 시 콤보 리셋

### 3.3 생명 & 패널티
- 시작 생명: **3**
- 슬랑이/왁뿌볼을 놓침 → 생명 -1
- 얼음 슬라이스 → 생명 -3 (한 방에 게임 오버) + 화면 흔들림 + 콤보 리셋
- 생명 0 → 게임 종료
- (확장) 생명 회복 아이템으로 추후 변형 가능

---

## 4. 연출 (Juice)

> 이 게임의 핵심은 "자르는 손맛". 시청각 피드백에 가장 공들일 부분.

### 4.1 비주얼
- **스쿼시 & 스트레치**: 튀어오를 때 납작, 정점에서 늘어남
- **슬라이스 순간**: 오브젝트가 두 조각으로 분리되어 각각 물리 적용
- **파티클**: 슬랑이는 젤리 방울, 왁뿌볼은 왁스 파편, 얼음은 성에 입자
- **궤적(Trail)**: 손가락 드래그를 따라 빛나는 칼날 트레일
- **화면 피드백**: 콤보 시 가벼운 줌/플래시, 얼음 시 흔들림

### 4.2 사운드 (★ 이 게임의 주인공)
> 컨셉 방향: **ASMR 감각 위주.** 효과음 하나하나가 메인 콘텐츠. BGM은 최소.

- 오브젝트별 고유 효과음 (말랑 / 퍽 / 캉) — 질감이 확실히 다르게
- 같은 오브젝트도 여러 변형 샘플을 랜덤 재생해 단조로움 방지
- 콤보 상승 시 음 높이 단계적 상승 (점점 신나는 느낌)
- 슬라이스 손맛을 위해 사운드 지연 최소화 (터치 즉시 재생)
- 모바일 햅틱(진동)과 사운드 동기화로 촉감 강화
- 배경음: **아주 잔잔한 앰비언트만 최소한으로** (또는 무음 옵션 제공)
- **소스**: Freesound.org, Zapsplat 무료 효과음 또는 AI 효과음 생성(ElevenLabs 등)

## 5. UI/UX 흐름

### 5.1 화면 구성
```
[시작 화면] → [게임 플레이] → [결과 화면] → (재시작 / 메인)
```

### 5.2 플레이 화면 HUD
- 상단 좌측: 점수
- 상단 우측: 생명(하트) 또는 남은 시간
- 중앙 하단: 오브젝트 등장 영역
- 콤보 시 화면 중앙에 콤보 수 팝업

### 5.3 결과 화면
- 최종 점수
- 최고 콤보
- 최고 기록(베스트 스코어)
- 다시하기 / 공유 버튼

---

## 6. 확장 방향

- **게임 모드**: 우선 단일 모드(생명 3개 서바이벌)로 출시 → 추후 타임어택 추가
- **신규 오브젝트**: 황금 슬랑이(레어 고득점), 멀티 슬랑이(연쇄 폭발)
- **스킨/테마**: 슬랑이 색상·표정 커스터마이징
- **데일리 챌린지**: 매일 다른 목표
- **리더보드**: 친구/전체 랭킹
- **햅틱**: 모바일 진동으로 촉감 강화
