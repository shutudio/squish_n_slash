# 🧀 Squish & Slash

> 말랑한 슬랑이를 자르고, 왁뿌볼을 부수는 ASMR 슬라이스 게임

**▶ 지금 플레이하기: https://shutudio.github.io/squish_n_slash/**

폰 브라우저에서 "홈 화면에 추가"를 누르면 앱처럼 설치됩니다. (PWA 지원)

## 🎮 게임 소개

화면 아래에서 통통 튀어오르는 말랑이들을 손가락으로 슬라이스! 단, 딱딱이 얼음은 절대 자르면 안 됩니다.

| 오브젝트 | 액션 | 점수 |
|---|---|---|
| 🧀 슬랑이 | 자른다 — 찰방~ 쫀득~ | +10 |
| 🎨 왁뿌볼 | 부순다 — 파바박! (실제 왁뿌볼 녹음) | +20 |
| ✨ 황금 슬랑이 | 레어! | +50 |
| 🎊 멀티 슬랑이 | 새끼 슬랑이 3마리 팡팡! | 연쇄 보너스 |
| 🧊 딱딱이 얼음 | 피한다! | 💔💔💔 |

## ✨ 주요 기능

- **ASMR 사운드** — 직접 녹음한 왁뿌볼 크랙 사운드 + Web Audio로 실시간 합성한 슬라임 질감음. 콤보 멜로디는 설정에서 끄고 순수 ASMR로 즐길 수 있어요
- **콤보 & 피버타임** — 한 번에 4개 이상 자르면 4초간 점수 2배 + 노 패널티 피버
- **가챠 & 도감** — 게임 점수의 10%가 코인으로 적립. 1회/10회 뽑기로 캐릭터 16종 수집 (일반~전설). 모은 캐릭터는 게임에 직접 등장하고, 에픽·전설은 보너스 점수까지!
- **2가지 모드** — 기본(생명 3개 서바이벌) / 타임어택(60초, 얼음은 −5초)
- **업적 11종 + 데일리 출석 보상** — 코인 벌 구멍이 많아요
- **인터랙티브 튜토리얼** — 첫 실행 시 직접 해보며 배우는 4단계 튜토리얼
- **마스코트** — 로비의 치즈 슬랑이를 누르면 또잉또잉 합니다 🍮
- **점수 카드 공유** — 결과 화면에서 예쁜 점수 이미지를 만들어 공유

## 🛠 기술

- **단일 `index.html` 파일** — 프레임워크 없이 Vanilla JS + Canvas 2D
- **Web Audio API** — 효과음 실시간 합성 (노이즈 필터·주파수 챠프로 슬라임/왁스/얼음 질감 표현) + 녹음 샘플 재생
- **스프라이트 캐시** — 오브젝트를 등장 시 1회만 렌더링해 모바일 60fps 유지
- **localStorage** — 코인·도감·업적·기록 저장 (서버 없음)
- **PWA** — 매니페스트 + 서비스워커로 설치/오프라인 지원
- **배포** — GitHub Pages
