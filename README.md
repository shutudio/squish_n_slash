# 🧀 Squish & Slash

## 🎮 게임 소개

화면 아래에서 통통 튀어오르는 말랑이들을 손가락으로 슬라이스! 단, 딱딱이 얼음은 절대 자르면 안 됩니다.

| 오브젝트 | 액션 | 점수 |
|---|---|---|
| 🧀 슬랑이 | 자른다 — 찰방~ 쫀득~ | +10 |
| 🎨 왁뿌볼 | 부순다 — 파바박! (실제 왁뿌볼 녹음) | +20 |
| ✨ 황금 슬랑이 | 레어! | +50 |
| 🎊 멀티 슬랑이 | 새끼 슬랑이 3마리 팡팡! | 연쇄 보너스 |
| 🧊 딱딱이 얼음 | 피한다! | 💔💔💔 |

## ✨ 주요 기능

- **ASMR 사운드** — 직접 녹음한 왁뿌볼 크랙 사운드 + Web Audio로 실시간 합성한 슬라임 질감음. 콤보 멜로디는 설정에서 끄고 순수 ASMR로 즐길 수 있어요
- **콤보 & 피버타임** — 한 번에 4개 이상 자르면 4초간 점수 2배 + 노 패널티 피버
- **가챠 & 도감** — 게임 점수의 10%가 코인으로 적립. 1회/10회 뽑기로 캐릭터 16종 수집 (일반~전설). 모은 캐릭터는 게임에 직접 등장하고, 에픽·전설은 보너스 점수까지!
- **2가지 모드** — 기본(생명 3개 서바이벌) / 타임어택(60초, 얼음은 −5초)
- **업적 11종 + 데일리 출석 보상** — 코인 벌 구멍이 많아요
- **인터랙티브 튜토리얼** — 첫 실행 시 직접 해보며 배우는 4단계 튜토리얼
- **마스코트** — 로비의 치즈 슬랑이를 누르면 또잉또잉 합니다 🍮
- **점수 카드 공유** — 결과 화면에서 예쁜 점수 이미지를 만들어 공유

## 🛠 기술

- **단일 `index.html` 파일** — 프레임워크 없이 Vanilla JS + Canvas 2D
- **Web Audio API** — 효과음 실시간 합성 (노이즈 필터·주파수 챠프로 슬라임/왁스/얼음 질감 표현) + 녹음 샘플 재생
- **스프라이트 캐시** — 오브젝트를 등장 시 1회만 렌더링해 모바일 60fps 유지
- **localStorage** — 코인·도감·업적·기록 저장 (서버 없음)
- **PWA** — 매니페스트 + 서비스워커로 설치/오프라인 지원
- **배포** — GitHub Pages

## 💜 제작 노트

이 게임은 **바이브 코딩(vibe coding)으로 제작되었습니다.** 기획서(PRD)를 기반으로 Claude와 협업하며, 짧은 피드백 루프로 빌드 → 플레이 테스트 → 개선을 반복하는 방식으로 개발했습니다. 주요 이터레이션은 다음과 같습니다.

- **사운드 디자인** — Web Audio API의 노이즈 필터 스윕, 주파수 챠프, ADSR 엔벨로프 파라미터를 반복 조정하며 슬라임의 점성·왁스 셸의 파열 질감을 합성으로 구현. 이후 실제 왁뿌볼을 녹음한 샘플을 RMS 기반 이벤트 검출로 분할·정규화하여 적용
- **슬라이스 판정** — 드래그 궤적의 선분-원 거리 판정으로 슬라이스를 검출하고, 절단선을 오브젝트 중심 고정이 아닌 실제 궤적의 통과 위치(부호 있는 수직 거리)로 계산해 매번 다른 단면이 나오도록 개선
- **분리 연출** — 절단면을 울퉁불퉁한 폴리라인 클리핑으로 처리하고, 점성 감쇠·멜팅 변형·연결 가닥(strand)을 더해 과일이 아닌 슬라임의 물성을 표현
- **성능 최적화** — 매 프레임 그라디언트를 재생성하던 렌더링을 스프라이트 캐시(등장 시 1회 오프스크린 렌더) 방식으로 전환하고 DPR을 제한하여 모바일에서 안정적인 프레임 확보
- **게임 루프 설계** — 가챠 경제(코인 적립률·천장 없는 확률 테이블·중복 환급), 피버타임 리스크 제거, 업적 보상 등 리텐션 구조를 단계적으로 추가

기획 문서는 [`Squish_and_Slash_prd.md`](./Squish_and_Slash_prd.md)에서 볼 수 있습니다.
